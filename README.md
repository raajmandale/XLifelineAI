## 🚀 Quickstart

### 1. Clone

git clone https://github.com/raajmandale/XLifelineAI.git  
cd XLifelineAI  

---

### 2. Setup environment

python -m venv .venv  
.venv\Scripts\activate  

---

### 3. Install dependencies

pip install -r requirements.txt  
pip install -e .  

---

### 4. Run demo

python examples/resurrection_demo.py
